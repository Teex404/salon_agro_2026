# 🌱 Salon de l’Agronomie 2026 – Event Management System

Système web développé pour gérer le **Salon de l’Agronomie 2026**.  
La plateforme permet de gérer les étudiants, les stands, les événements et la validation des participations grâce à un **système de scan QR code sécurisé**.

Ce projet a été développé avec **PHP, MySQL, Bootstrap et JavaScript** dans un environnement **XAMPP**.

---

# 🚀 Fonctionnalités principales

### 🎓 Gestion des étudiants
- création des profils étudiants
- génération de tokens QR uniques
- activation / désactivation des comptes

### 🏢 Gestion des stands
- validation des visites
- prévention de la fraude
- scan par les contrôleurs de stands

### 🎤 Gestion des événements
- validation de participation
- scan de présence
- suivi des activités

### 📊 Tableau de bord administrateur
- suivi des participations
- gestion des utilisateurs
- statistiques

---

# 🧠 Concept du système

Le système repose sur un **token unique associé à chaque étudiant**.

Chaque activité (stand ou conférence) est validée par un **scan effectué par les contrôleurs**, ce qui évite :

- la fraude
